# Housing-prices

A simple cli that finds the latest Norwegian housing prices in a given location. The results are taken from finn.no.

### 🚀 Usage

```
$ node cli.js <options>
```

### 🌎 Options

```
--location, -l  Housing location
```

### 📚 Examples

```
$ node cli.js --location 1.20061.20512
    🏡 4 500 000
    🏡 3 290 000
```
